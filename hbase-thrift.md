# Thrift

## Install Thrift Compiler

To get started, [download](https://thrift.apache.org/download) a copy of Thrift.

generate the source from a thrift file run:

```shell
> thrift --gen <language> <Thrift filename>
```

Show help

```shell
> thrift -help
```

## HBase Thrift

```sh
> thrift --gen go
```
